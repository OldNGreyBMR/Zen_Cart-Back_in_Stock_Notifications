Installation

Installation of the module has been made as simple as possible and shouldn't take long for any stores which have not already modified any of the template files which this module has to modify.

If any of the template files this distribution have already been modified (because of another third party module being installed) then the changes for the respective files should be applied manually by copying across the appropriate sections of code required for the Back In Stock Notifications module. These sections are clearly marked (with “// BEGIN” and “// END” comments) within the sample files provided in this distribution.

Installation Overview

  1. Copy the new files from the distribution into your Zen Cart installation.
  2. Copy the updated template files from the distribution into your Zen Cart installation or modify their equivalents in the store's existing template files.
  3. Create the database table & add the admin configuration options by running the provided SQL script.

  2. Updated Template Files For Zen Cart 1.3.x

    Several of the standard Zen Cart template files require modifications to work with this module.

    These modifications have been made to “fresh” copies of the template files from a “clean” installation of Zen Cart and are included as sample files in this distribution.

    • If these template files haven't already been modified for the store, the sample files included in this distribution can be copied into the store template's templates directory and will then override the standard versions of the files included in the Zen Cart distribution.

    • If any of these template files have already been modified for the store, then the changes necessary for the Back In Stock Notifications module to function must be made to the store's modified file(s). The changes are clearly marked within the sample files in the distribution and should be easy enough to copy across, especially if a “diff” software package (such as the free WinMerge) is used.

    The sample modified template files can be found within the template_override_files folder in the distribution. They are:

    • includes/templates/YOUR_TEMPLATE/templates/tpl_account_default.php
    • includes/templates/YOUR_TEMPLATE/templates/tpl_product_info_display.php

  3. Create the Database Table & Add the Admin Configuration Options

    A SQL script, mysql.sql, is provided with this module to create the database table in which the module stores subscriptions and to add the configuration options for the module in the Zen Cart admin.

    • You can go to Tools > Install SQL Patches in the Zen Cart Admin and run the file through that.

    • Or you can use software like PHPMyAdmin to run the SQL script (Don't forget to check the name of the tables. If you use a prefix for your table names you should make sure that you put this prefix in front of the table names). If using this method please note that some double backslashes may need to be removed in the source.
