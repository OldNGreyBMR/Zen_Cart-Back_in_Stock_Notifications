Upgrading to Version 2.2.4 from Version 2.2.3

The changes between these versions are minor and easy to make:

  1. Copy the new files across

    Overwrite all existing files with the updated/new files from version 2.2.4.

    They already have the correct folder structure, so a single drag-and-drop of the folders/files in the files folder into the root directory for the store will put the files in the correct locations; there's no need to copy the files individually - that often results in files being missed out!

Upgrading to Version 2.2.4 from Version 2.2.2/2.2.1

The changes between these versions are minor and easy to make:

  1. Copy the new files across

    Overwrite all existing files with the updated/new files from version 2.2.4.

    They already have the correct folder structure, so a single drag-and-drop of the folders/files in the files folder into the root directory for the store will put the files in the correct locations; there's no need to copy the files individually - that often results in files being missed out!

  2. Adjust a custom template file

    If you have modified the file includes/templates/YOUR_TEMPLATE/templates/tpl_account_default.php for the template you are using, it must be updated with the code from the sample file in 2.2.4.

    • Copy the relevant updated source to your modified version.

    If you haven't modified this file, you can simply overwrite the file for the template you are using with the file from the template_override_files folder in the distribution.

    • includes/templates/YOUR_TEMPLATE/templates/tpl_account_default.php

Upgrading to Version 2.2.4 from Version 2.2.0

The changes between these versions are minor and easy to make:

  1. Copy the new files across

    Overwrite all existing files with the updated/new files from version 2.2.4.

    They already have the correct folder structure, so a single drag-and-drop of the folders/files in the files folder into the root directory for the store will put the files in the correct locations; there's no need to copy the files individually - that often results in files being missed out!

  2. Adjust two custom template files

    If you have modified the file includes/templates/YOUR_TEMPLATE/templates/tpl_account_default.php or the file includes/templates/YOUR_TEMPLATE/templates/tpl_product_info_display.php for the template you are using, either/both must be updated with the code from the sample files in 2.2.4.

    • Copy the relevant updated source to your modified version of either/both files.

    If you haven't modified one (or either) of these files, you can simply overwrite the appropriate file(s) for the template you are using with the respective file from the template_override_files folder in the distribution:

    • includes/templates/YOUR_TEMPLATE/templates/tpl_account_default.php
    • includes/templates/YOUR_TEMPLATE/templates/tpl_product_info_display.php
