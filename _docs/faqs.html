Frequently Asked Questions (FAQs)

  1. I have already modified some of the template files for my site. How do I add the Back In Stock Notifications code?

    Each change in the template files required for the Back In Stock Notifications module to work is clearly marked in the sample versions of the files.

    Changes begin with a comment such as:

    // BEGIN BACK_IN_STOCK_NOTIFICATIONS 1 of x

    And end with a comment such as:

    // END BACK_IN_STOCK_NOTIFICATIONS 1 of x

    Where ‘x’ is the number of code changes within the current file which must be included for the Back In Stock Notifications module to use that file correctly.

    • Each change should be copied and pasted into your modified template file at the exact same place it is present within the modified sample files.
